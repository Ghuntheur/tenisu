# TENISU

Le projet est accessible en ligne à [cette adresse : https://tenisu.netlify.app/](https://tenisu.netlify.app/)

### Installation

- Lancer le serveur de dev : `npm run dev`
- Lancer les tests : `npm run test`
- Build le projet : `npm run build`
